6.5 Feedback Collection

Scenario: A survey system stores feedback messages in feedback.txt. Write a program to count total feedback messages and find the longest one. Save results in a new file feedback-summary.txt.

Algorithm:

1. Open feedback.txt in read mode.

2. Initialize counter to zero.

3. Read all feedback messages into a list.

4. Count total feedback messages.

5. Find the longest feedback message using length comparison.

6. Store results in a string summary.

7. Open feedback-summary.txt in write mode.

8. Write the summary into the new file.

9. Close the file.

10. Display "Feedback Summary saved."
